building a neural network architecture
1. membuat layer
layer_1 = Dense(units=3, activation='sigmoid')
layer_2 = Dense(units=1, activation='sigmoid')

2. membuat model
tujuan untuk memberitahu dan merangkai layer-layer yang telah dibuat
sehingga kita tidak perlu memasukkan hasil layer_1 ke layer_2, dst.
model = Sequential([layer_1, layer_2])

untuk langkah 1 dan 2 dapat disingkat seperti ini:
model = Sequential([
    Dense(units=3, activation='sigmoid'),
    Dense(units=1, activation='sigmoid')
])

(contoh kasus ini adalah neural network pada pemanggangan kopi, 
parameter 1 = suhu, parameter 2 = lama panggang(menit))
hasil = 1 = baik
0 = tidak baik
hasil akan muncul kira-kira seperti ini 
x = np.array([
    [200, 17],
    [120, 5],
    [425, 20],
    [212, 18],
])
y = np.array([1,0,0,1])

3. melatih jaringan saraf
model.compile(...)

fungsi dibawah berfungsi untuk memberitahu tensorflow bahwa terdapat 
model dengan layer-layer yang telah dirangkai dan akan dilatihkan pada data x dan y.
model.fit(x,y)

4. prediksi hasil
model.predict(x_new)
